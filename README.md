# Mi Primer Script de Supervivencia Financiera

Este es un script de Python simple creado como parte del curso "Python para la Innovación" por CUGDL y GDG Jalisco.

## Descripción

El script `Semana 1.py` actúa como un asistente financiero básico. Ayuda a los usuarios a realizar un seguimiento de sus gastos semanales y a comprender su situación financiera al final de la semana.

## ¿Cómo funciona?

1.  **Entrada de datos**: El script solicita al usuario que ingrese su presupuesto semanal y sus gastos en diferentes categorías (transporte, comida, copias y libros).
2.  **Cálculos**: Calcula el gasto total y el balance final restando los gastos del presupuesto.
3.  **Resumen y recomendaciones**: Muestra un resumen de las finanzas de la semana y ofrece una recomendación basada en si el usuario tiene un superávit, un presupuesto ajustado o un déficit.

## ¿Cómo usarlo?

1.  Asegúrate de tener Python instalado en tu sistema.
2.  Ejecuta el script desde la terminal con el comando:
    ```bash
    python "Semana 1.py"
    ```
3.  Sigue las instrucciones en pantalla para ingresar tus datos financieros.
4.  ¡Recibe tu resumen y recomendaciones financieras!
